# PROMPT 0 — QUALIFICATION GATE

Цель: проверить готовность входа для запуска полного R&D конвейера.

Сделай:
1. Проверку минимальной полноты intake.
2. Вывод статуса: `ГОТОВО` или `НЕ ГОТОВО`.
3. При `НЕ ГОТОВО`:
   - верни только диагностический вывод,
   - задай до 7 точных вопросов для закрытия пробелов.
4. Пустые поля маркируй `MISSING/НЕ ПРЕДОСТАВЛЕНО`.
5. Используй только переданные evidence.

Никаких внешних источников.
